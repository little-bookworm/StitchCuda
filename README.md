# StitchCuda

## 1 Requirement
This codebase has been developed with cuda-11.1, OpenCV-3.2.0 and YAML ...

## 2 Installation
### 2.1 Install modules *StitchCuda*
```
git clone git@github.com:little-bookworm/StitchCuda.git
cd StitchCuda
bash build.sh
```
